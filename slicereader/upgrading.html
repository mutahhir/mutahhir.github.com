Notice for people upgrading Slicereader

I'm sorry, but this update will empty your existing reading list.
It's my fault for not checking this before, but I realized that the articles were being under the wrong name after the initial launch. Actually, that release was pretty sloppy because it wasn't a release at all. I didn't expect anyone to use it. Anyhow, the point remains …

I'm sorry!

However, there is a fix, and it's not hard. You can get your reading list back, just like it was before. I'm going to try and detail the steps as straightforwardly as I can, but in case you have any problems, contact me on twitter or send me an email

Ok, so here we go

  • In your mac there's a folder with the path 

    ~/Library/Application Support/jamfilled.Brake
    navigate to that.

    If you don't know how to do that, it's easy: Select the finder icon on your dock (the left most one). Then, either press Shift-Command-G or use the main menu Go > Go to Folder…

    Copy and paste 

    ~/Library/Application Support/jamfilled.Brake
    there.

  • You will see a file called 
    Brake.storedata
    . Copy that file (Cmd-C, or right-click > Copy).
  • Run the new, updated Slicereader once. If you haven't downloaded it, do it now.
  • Using the same method explained in Step 1, go to 
    ~/Library/Application Support/me.mthr.slicereader
  • Paste the 
    Brake.storedata
    file here
  • There will be a file in this directory already called 
    slicereader.storedata
    . Delete that file.
  • Now rename 
    Brake.storedata
    to 
    slicereader.storedata
    and you're done
  • Re-open slicereader. Viola! (Hopefully :))
